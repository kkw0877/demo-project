# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

카카오톡 나만의 채팅방 메모 분석기 — a web app that lets users upload a KakaoTalk CSV export, then uses Claude to analyze the memos and return structured insights (summary, action items, topics, recent highlights).

## Running the app

Two servers must run simultaneously:

```bash
cd backend && npm run dev
cd frontend && npm run dev
```
